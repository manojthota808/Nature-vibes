# Nature Studios - Photography Gallery

Welcome to **Nature Studios**, a personal photography portfolio website showcasing the beauty of the natural world and cherished moments.

## üåü Overview

This website serves as a digital gallery for my photography journey. It features two distinct collections:
*   **Nature Gallery**: Capturing the essence of nature, from majestic mountains to tranquil forests and serene deserts.
*   **Yeshu Gallery**: A special collection dedicated to baby photos.

The site is designed with a focus on aesthetics, offering a premium and immersive user experience with smooth transitions, an interactive slider, and a responsive layout.

## ‚ú® Features

*   **Dynamic Gallery Switching**: Seamlessly switch between "Nature" and "Yeshu" galleries without reloading the page.
*   **Interactive Slider**: A full-screen, touch-friendly image slider on the homepage.
*   **Lightbox Modal**: Click on any image to view it in a high-quality lightbox with navigation.
*   **Pagination**: Organized browsing for large image collections (specifically in the Nature Gallery).
*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
*   **About Me Section**: A personal introduction to the photographer behind the lens.
